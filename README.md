## Install

### Installing server
```sh
cargo install web-queue-server
```

### Installing proxy
```sh
cargo install web-queue-proxy
```

## Usage

### Running server
```sh
ADDR=0.0.0.0:8080 web-queue-server
```

### Running proxy
```sh
ADDR=0.0.0.0:8081 SHARDS=http://127.0.0.1:8080 web-queue-proxy
```
